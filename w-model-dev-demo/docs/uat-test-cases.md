## 用例列表

### UAT-001

- 标题：用户注册成功
- 优先级：高
- 关联需求：REQ-001
- 测试场景：新用户提交合法用户名+密码注册

**前置条件**
- 服务已启动，监听 3000 端口
- 用户名 `alice` 未被注册

**测试步骤**

| 步骤 | 操作 | 输入 | 预期输出 |
|---|---|---|---|
| 1 | POST /api/auth/register | `{"username":"alice","password":"Passw0rd!"}` | HTTP 201；body 含 `userId` |

**预期结果**
返回 201 状态码，响应体包含 `userId` 字段（非空字符串）。

**执行状态**
- [x] 待执行
